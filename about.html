About This Dashboard

A fan-built tracking dashboard for the 2026 FIFA World Cup.

This dashboard was created to track the 2026 FIFA World Cup — the largest World Cup in history, featuring 48 teams across 3 host nations (USA, Canada, Mexico) and 16 venues.

🏆 About the 2026 FIFA World Cup

The 2026 FIFA World Cup will be held from June 11 to July 19, 2026, jointly hosted by the United States, Canada, and Mexico. This is the first World Cup to feature 48 participating nations, expanded from the previous format of 32 teams.

📊 Dashboard Features

  • Home: Tournament overview, countdown timer, key dates, and host nation info
  • Groups: All 12 group standings (A–L) tracking points, goals, and goal difference
  • Schedule: All 72 group stage matches with date, time, venue, and filter options
  • Teams: All 48 participating nations organized by confederation
  • Bracket: Knockout stage bracket from Round of 32 through the Final
  • Stats: Golden Boot top scorers, discipline card leaders, and sortable team stats

🔧 Technical Details

This is a static HTML/CSS/JavaScript dashboard with no backend dependencies. It is inspired by the NFL Stats Dashboard and similar fan dashboards. Core tournament structure lives in app.js, while the live schedule, results, and stat-event feed is generated into scores.js. Data refreshes automatically every 3 hours via a GitHub Actions workflow that fetches the latest World Cup matches from the football-data.org API and commits them to scores.js, with manual re-runs available from the repository Actions tab whenever an immediate refresh is needed.
